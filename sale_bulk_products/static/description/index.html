BULK PRODUCTS

This module purpose is to add multiple line item in single view.
After installing this module you can view "Bulk Products" button below order lines. when creating the quotation you want to select the products one by one but this module will help you to select the multiple products suddenly.your process will increases and also time will save more.you can create multiple sale orders quickly.

Step 1:

Creating a quotation.if you need multiple products in line items.you can click "Bulk Products" button.

Step 2:

Once button clicked you can see wizard and choose your products then click "confirm" button.

Step 3:

In this Wizard you can search more products and filter your product items

Step 4:

Selected product items shows in sale order lines.
