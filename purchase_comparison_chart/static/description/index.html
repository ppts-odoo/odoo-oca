PURCHASE COMPARISON CHART

This Module is Purchase agreement type and comparison create the agreement to the vendor's for product materials & costs. we can easily analysis the details and also view comparison chart on Website & Excel sheet.

Step 1:

In Purchase there is 'Purchase Agreement Type' menu for creating the agreement types and config.

In same Purchase 'Purchase Agreements' menu to create the agreements for vendors and select agreement type click "Confirm" button to create agreement.

Step 2:

After creating the agreement and you need to create the purchase quotation based on the vendors.

Step 3:

After creating the RFQs/Orders click the 'Purchase Comparison' button to view.

Step 4:

Get accurate comparison

Get the chart you need to make smarter decisions.

Analysis the chart and click 'Back to Purchase Requistion' button

Step 5:

After Creating the RFQs/Orders if you need XLS report then click 'Print XLS' button to view.

Step 6:

Now you can see the Excel Sheet Report.
