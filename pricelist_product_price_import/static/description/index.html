PRICELIST PRODUCT PRICE IMPORT


This module allows you to import products to line items based on product category. The module used to import the products by one click and also you update the line items product price in one click




Step 1:


Choose categories you required to import product to line items.



Enable Include Pricelist to include products in pricelist, Disable to exclude products in pricelist



Step 2:

Once you selected the category to choose where it to be applied on.



Step 3:

Set global used to set prices for all products, irrespective of seleced categories.



Set product category to create new category based on product category you have selected



Set the Product to import products from selected categories to the line items



Set the Product Variant to import products from selected categories to the line items



Step 4:

Similar like the pricelist items, you can choose options and price computation method.



Step 5:

Set Minimum Quantity for all the product inline items, Choose start date and end date for the availability of the product in line items. On compute price, choose fix price to set a fixed price for all the products in the line items.



Choose percentage to set a discount price for all the products in the line items.



Choose Formula to compute percentage price with the rounding method and also set minimum-maximum margin



If you want to set a new formula method to compute product price then choose the public price
