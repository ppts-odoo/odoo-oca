Website Product Name

Overview

In odoo default the name given in product will be displayed in website. This module helps to display seperate name in website.

Configuration

Step 1:

Go to Website -> Products -> Products -> select any product.

Step 2:

Click on Edit and add a name in Website product name field to display in website.

Website View

The name given in website product name field will be displayed in website.
