Dropshipping Extended

In this module Drop Shipping Extended you can change Warehouse Delivery order to Dropship(Vendor Delivery Order) and Dropship (Vendor Delivery Order) to Warehouse Delivery Order.

Step 1:

Configure Default Destination Location in Warehouse

Step 2:

In Delivery Order Under Action we can choose Create Dropship to Deliver product from Vendor./i>

Step 3:

Choose Boolean Fields in wizard line item.After Confirmation it will create a Purchase order for Selected Product.After Confirmation of Purchase Order and related Delivery Order Product will deliver from Vendor Location.

Step 1:

Instead of Delivering product from vendor,If need to deliver product from warehouse use create delivery order method under action

Step 2:

Choose Boolean Fields in wizard line item.After Confirmation it will create a Delivery order for Selected Product.Now the Product will be delivered from Warehouse instead of Vendor Location.
