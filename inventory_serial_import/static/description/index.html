Import Serial Numbers on Delivery Orders

Importing lot numbers for the particular product in delivery orders of inventory.

Step 1:

In "Lots/Serial Numbers" menu under inventory control, add some "serial number" for products.

Step 2:

Select a delivery order and click on the list icon present in order lines.

Step 3:

Upload a "csv" file and click on validate button.

Step 4:

The serial numbers provided in the "csv" file will be updated in the order lines.
