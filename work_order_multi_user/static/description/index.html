WORK ORDERS IN MULTI USER

This module purpose is to work multiple users in work order.
In odoo when one user block other user also blocked in work orders but this module have a option to change this option when one user block other users can work in work orders.This module is mainly for multi users in work orders and you can block the particular user and also you can view the blocked users in work orders.this module you can use only in "Enterprise Version".

Step 1:

Create the new "MO" and also create the work orders for the product materials.

Step 2:

In work order you can click the "Start Working" button for the product.

Step 3:

After started the working if you want to block the user and click the "Block" button.

Step 4:

In "Blocked Users" you can see the blocked user details and click "Unblock" button for unblock the user.

Step 5:

You can login another one user to start work orders click the "Continue Production" button.

Step 6:

You can see the multi users in blocked users with detail.
