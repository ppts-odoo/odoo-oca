Bulk Products to Cart

Add bulk products to cart quickly

Overview

Bulk Products to Cart helps to the user to add the multiple products easily and can proceed the orders as quickly and also saves the user time.

Configuration

Install the Module "Bulk Prodcuts to Cart" in order to use bulk import order option in website.

Bulk Prodcuts to Cart

  • Copy - Paste the product name from the website
  • Search to prepare the order
  • Update the quantity and refresh to see the updated order
  • Click button to add products to cart

Screenshots

After installing the "Add Bulk products to Cart" application then the features will be visible in website.

Click on the menu and copy and paste the product name from the website. The format will be Product Reference # or Product Name, Quantity, Your comment

Click on the Search button to see the order with amount

If you would like to check the price after changing the quantity, change the quantity and click on the Refresh button to see the updated order details with price

After all the verification. Click on the Create Order button to add the products to the cart.

Now the added products will be added under the cart. Now you can proceed further.
